## Критерии оценки

### 1. Визуальный дизайн (по скриншотам)
- Соответствует ли дизайн позиционированию «тихая роскошь» (quiet luxury)?
- Цветовая палитра: гармония, контрастность, читаемость
- Типографика: уместность выбора шрифтов, иерархия, ритм
- Композиция: баланс пустого пространства, визуальный вес элементов
- Сравни с топовыми detailing/automotive-студиями

### 2. UX и информационная архитектура
- Логичность порядка секций (воронка: внимание → интерес → доверие → действие)
- Читаемость и сканируемость контента
- CTA-стратегия: расположение, формулировки, визуальный вес
- Баланс между эстетикой и конверсией

### 3. Техническая архитектура
- Оцени решение «один HTML-файл без зависимостей» — плюсы и минусы
- Scroll-driven анимация: подход, производительность, UX
- Lazy-loading стратегия для 64 кадров
- CSS-система: переменные, компоненты, responsive подход
- JavaScript: модульность, качество кода

### 4. Креативные решения
- AI-пайплайн (фото → Google Flow → кадры → scroll-анимация): оригинальность и уместность
- Hero-секция: многослойность (фото + оверлеи + блик)
- BA-слайдер: реализация и UX
- Fade-in анимации: уместность и тайминги

### 5. SEO и мета-данные
- Schema.org разметка для локального бизнеса
- Open Graph теги
- Семантическая структура HTML

### 6. Слабые стороны
- Что конкретно нужно доработать перед запуском?
- Какие архитектурные решения ты бы изменил?
- Что не соответствует стандартам premium-сегмента?

## Формат ответа

Для каждого критерия:
- Оценка: X/10
- Сильные стороны: [что сделано хорошо]
- Слабые стороны: [что нужно улучшить]
- Рекомендации: [конкретные действия]

В конце:
- Общая оценка: X/10
- Топ-3 сильных стороны проекта
- Топ-3 критических улучшения перед запуском
- Вердикт: готов ли проект для portfolio / production / обоих
