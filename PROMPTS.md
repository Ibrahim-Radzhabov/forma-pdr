## Промпт 1: Продолжение работы над проектом FORMA

> Используй этот промпт, когда передаёшь проект новому агенту для доработки.
> Скопируй и вставь как системный промпт или первое сообщение.

```
Ты работаешь над проектом FORMA — лендинг-страница для премиальной студии
беспокрасочного удаления вмятин (PDR) в Москве.

## Контекст проекта

Это single-page лендинг в одном HTML-файле (forma_landing_v3.html, ~9 MB).
Без фреймворков, без сборщиков — чистый HTML + CSS + Vanilla JS.
Все изображения встроены инлайн в base64.

Дизайн-позиционирование: «тихая роскошь» (quiet luxury) — как сайты
часовых мастерских или премиальных detailing-студий. Тёмная палитра
(#0a0a0a) + приглушённое золото (#c9a96e) + Georgia serif для заголовков +
Arial sans-serif для текста.

## Структура страницы (порядок секций)

1. Preloader — прогресс загрузки кадров canvas-анимации
2. Nav — фиксированная, blur-фон, золотая граница
3. Hero — fullscreen, фоновое фото кузова с LED-полосами, затемнённый
   оверлей слева, золотой радиальный блик
4. Stats — 4 колонки (1200+ вмятин, 97% без покраски, 8 лет, рейтинг 4.9)
5. Frame Sequence — scroll-driven анимация на 64 кадрах: ровная
   поверхность → вмятина нарастает → пик → ремонт → ровная поверхность.
   Кадры из видео Google Flow, lazy-load батчами по 8
6. Before/After — BA-слайдер с drag (mouse + touch)
7. Services — сетка 3x2, 5 услуг + CTA-карточка
8. Why — преимущества (2 колонки: список + визуальный блок с SVG)
9. Process — 5 шагов с соединительной линией
10. Gallery — сетка 3x2, hover-эффекты
11. Reviews — 3 карточки отзывов
12. Contacts — информация + форма + Яндекс.Карта (iframe, grayscale)
13. Footer

## CSS-система

Переменные: --gold (#c9a96e), --gold2 (#e8c98a), --dark (#0a0a0a),
--dark2 (#111), --dark3 (#1a1a1a), --text (#e8e4df), --muted (#888).
Компоненты: .section-tag, .section-title, .btn-primary, .btn-ghost,
.fade-in, .fade-in-delay-{1,2,3}.
Responsive breakpoint: 900px.

## JavaScript (5 модулей)

1. Frame Sequence — массив frames[], loadFrame(), showFrame(), preloader
2. Scroll Handler — updateFrameSeq(), rAF-throttled
3. BA Slider — drag mouse+touch, clip-path
4. Smooth Scroll — навигация и CTA
5. Fade-in — Intersection через getBoundingClientRect

## Промпт 2: Создание аналогичного проекта для другой сферы

> Используй этот промпт для создания нового лендинга по шаблону FORMA.
> Замени [PLACEHOLDER] на реальные данные перед использованием.

```
Создай премиальный single-page лендинг для [НАЗВАНИЕ БИЗНЕСА] —
[ОПИСАНИЕ ДЕЯТЕЛЬНОСТИ, 1 предложение] в [ГОРОД].

## Техническое задание

Один HTML-файл, без фреймворков, без внешних зависимостей.
Все стили в <style>, весь JS в <script>. Чистый HTML5 + CSS3 + Vanilla JS.

## Дизайн-позиционирование

Стиль: тихая роскошь (quiet luxury). Тёмная цветовая палитра,
один акцентный цвет, классический serif-шрифт для заголовков,
чистый sans-serif для текста. Никаких ярких красок, анимированных
градиентов или «игривых» элементов. Референсы: сайты часовых
мастерских, премиальных ателье.

## Цветовая палитра

Задай CSS-переменные в :root:
- --accent: [АКЦЕНТНЫЙ ЦВЕТ, hex] (основной акцент — CTA, заголовки, иконки)
- --accent2: [СВЕТЛЫЙ АКЦЕНТ, hex] (hover, градиенты)
- --dark: [ОСНОВНОЙ ФОН, hex] (тёмный, почти чёрный)
- --dark2: [ФОН СЕКЦИЙ, hex] (чуть светлее, для чередования)
- --dark3: [ФОН КАРТОЧЕК, hex]
- --text: [ЦВЕТ ТЕКСТА, hex] (тёплый белый/кремовый)
- --muted: [ВТОРИЧНЫЙ ТЕКСТ, hex] (серый)

## Структура страницы (обязательные секции)

### 1. Nav (фиксированная)
- Логотип слева (текст, letter-spacing 6px)
- Меню справа (5 якорных ссылок, uppercase, 13px)
- Фон: полупрозрачный с backdrop-filter:blur(12px)
- Нижняя граница: 1px solid rgba акцентного цвета

### 2. Hero (100vh)
- Фоновое изображение (position:absolute, filter:brightness(0.35))
- Оверлей: linear-gradient слева направо (плотный → прозрачный) +
  radial-gradient акцентного цвета (мягкий блик)
- Контент (z-index:2): бейдж → h1 (clamp 42-88px) → подзаголовок →
  2 кнопки (primary + ghost)
- Акцентное слово в h1 выделено цветом через <em>

### 3. Stats (цифры)
- CSS Grid 4 колонки, разделители border
- 4 ключевых метрики бизнеса
- Числа: крупные (52px), акцентного цвета
- Подписи: uppercase, 12px, muted

### 4. Before/After (BA-слайдер)
- Два изображения наложены, clip-path для обрезки
- Перетаскиваемый разделитель (золотой) с круглой ручкой
- Поддержка mouse и touch событий
- Метки «ДО» / «ПОСЛЕ»

### 5. Services (услуги)
- CSS Grid 3 колонки, gap:2px
- Карточки: номер → заголовок → описание
- Hover: акцентная полоска сверху (::before, transform:scaleX)
- Последняя карточка — CTA со ссылкой

### 6. Why (преимущества)
- 2 колонки: список (иконка + заголовок + описание) + визуальный блок
- 4 ключевых преимущества
- Визуальный блок: тёмный фон с декоративными SVG-линиями

### 7. Process (шаги работы)
- CSS Grid 4-5 колонок
- Горизонтальная соединительная линия (::before)
- Каждый шаг: круг с номером → заголовок → описание

### 8. Reviews (отзывы)
- CSS Grid 3 колонки
- Карточки: звёзды → цитата (italic) → автор (аватар-инициал + имя)

### 9. Contacts
- 2 колонки: информация + форма
- Информация: телефон, мессенджеры, адрес, часы работы
- Форма: имя + телефон + сообщение + кнопка
- Яндекс/Google карта (iframe, filter:grayscale)

### 10. Footer
- Логотип + копирайт + описание
- Фон темнее основного

## UX-компоненты

### Fade-in анимации
- Класс .fade-in: opacity:0, translateY(30px)
- Класс .visible: opacity:1, translateY(0)
- JS: проверка видимости через getBoundingClientRect на scroll/resize
- Задержки: .fade-in-delay-1 (0.1s), -2 (0.2s), -3 (0.3s)

### Responsive (breakpoint 900px)
- Nav: скрыть меню
- Padding: 60px → 24px
- Grid: схлопывать колонки (4→2, 3→1, 2→1)
- BA-слайдер: уменьшить высоту
- Footer: flex-direction column

## SEO

### Мета-теги:
- <title>, <meta name="description">, Open Graph (og:title, og:description,
  og:type, og:locale)

### Schema.org (JSON-LD):
- @type: [ТИП БИЗНЕСА — CafeOrCoffeeShop, BeautySalon, AutoRepair и т.д.]
- Поля: name, description, url, telephone, address, openingHours,
  aggregateRating, priceRange

### Favicon:
- Инлайн SVG: квадрат с цветом фона + первая буква названия акцентным цветом

## Правила

- Никаких внешних CDN, npm, Google Fonts (если нужен кастомный шрифт —
  @font-face с инлайн base64 или системные шрифты)
- Все стили в одном <style>, весь JS в одном <script>
- Изображения: base64 инлайн или плейсхолдеры с комментариями
- Код чистый, с комментариями-разделителями между секциями
- Чередование фона секций: --dark и --dark2
- Все интерактивные элементы должны работать на touch-устройствах

## Контент для заполнения
