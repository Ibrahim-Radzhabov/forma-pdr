# FORMA -- студия безпокрасочного ремонта вмятин

Лендинг премиальной PDR-студии в Москве. Тёмная тема, scroll-анимация процесса ремонта на Canvas, before/after-слайдер. Один HTML-файл, без сборки и фреймворков.

## Стек

- HTML5 (семантическая разметка, Schema.org)
- CSS3 (кастомные свойства, медиа-запросы, анимации)
- Vanilla JS (Intersection Observer, Canvas API, touch-события)
- Canvas-анимация на 64 кадрах (base64 JPEG)

## Особенности

- **Scroll-driven анимация** -- 64 кадра процесса PDR на Canvas, пользователь управляет прокруткой
- **Before/After слайдер** -- сравнение до и после ремонта с поддержкой мыши и touch
- **Тёмная премиальная тема** -- приглушённое золото на чёрном, serif-заголовки, многослойные оверлеи
- **SEO** -- Schema.org (AutoRepair), Open Graph, мета-теги для Яндекса
- **Адаптивная вёрстка** -- корректное отображение от мобильных до десктопа
- **Нулевые зависимости** -- никаких npm-пакетов, бандлеров и фреймворков

## Запуск

Открыть `forma_landing_v3.html` в браузере. Для удобства разработки можно использовать Live Server.

```

├── frames/                 # исходные кадры анимации (JPEG)
│   ├── frame_000.jpg
│   ├── ...
│   └── frame_063.jpg
```
